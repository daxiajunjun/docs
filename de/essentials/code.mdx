## Inline-Code

Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, umschließen Sie es mit Backticks (`).

```
Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, umschließen Sie es mit Backticks (`).
```

## Codeblöcke

Verwenden Sie [abgegrenzte Codeblöcke](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie den Code in drei Backticks einschließen und die führenden Backticks mit der Programmiersprache Ihres Snippets versehen, um Syntaxhervorhebung zu erhalten. Optional können Sie auch den Namen Ihres Codes nach der Programmiersprache schreiben.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```
