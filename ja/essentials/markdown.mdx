## タイトル

セクションヘッダーに最適です。

```md
## タイトル
```

### サブタイトル

サブセクションヘッダーに最適です。

```md
### サブタイトル
```

<Tip>

各**タイトル**と**サブタイトル**はアンカーを作成し、右側の目次にも表示されます。

</Tip>

## テキストフォーマット

ほとんどのマークダウンフォーマットをサポートしています。テキストをフォーマットするには、単に`**`、`_`、または`~`をテキストの周りに追加してください。

| スタイル      | 書き方                  | 結果                |
| ------------- | --------------------- | ------------------ |
| 太字          | `**太字**`             | **太字**            |
| 斜体          | `_斜体_`               | _斜体_              |
| 打ち消し線    | `~打ち消し線~`         | ~打ち消し線~        |

これらを組み合わせることができます。例えば、`**_太字と斜体_**`と書くと、**_太字と斜体_**のテキストが得られます。

上付き文字や下付き文字のテキストを書くにはHTMLを使用する必要があります。つまり、テキストの周りに`<sup>`または`<sub>`を追加します。

| テキストサイズ  | 書き方                      | 結果                      |
| --------------- | ------------------------ | ------------------------ |
| 上付き文字      | `<sup>上付き文字</sup>`     | <sup>上付き文字</sup>     |
| 下付き文字      | `<sub>下付き文字</sub>`     | <sub>下付き文字</sub>     |

## ページへのリンク

テキストを`[]()`で囲むことでリンクを追加できます。例えば、`[googleへのリンク](https://google.com)`と書くことで[googleへのリンク](https://google.com)が作成されます。

ドキュメント内のページへのリンクは、ルート相対である必要があります。基本的には、フォルダパス全体を含めるべきです。例えば、`[テキストへのリンク](/writing-content/text)`は、コンポーネントセクション内の「テキスト」ページにリンクします。

相対リンクは、`[テキストへのリンク](../text)`のように書いた場合、最適化しにくいため、開くのが遅くなります。

## 引用

### 単一行

ブロック引用を作成するには、段落の前に`>`を追加します。

> ドロシーは城の中の美しい部屋をいくつも通り抜けて彼女の後を追いました。

```md
> ドロシーは城の中の美しい部屋をいくつも通り抜けて彼女の後を追いました。
```

### 複数行

> ドロシーは城の中の美しい部屋をいくつも通り抜けて彼女の後を追いました。
>
> 魔女は彼女に鍋を掃除し、床を掃き、薪で火を絶やさないように命じました。

```md
> ドロシーは城の中の美しい部屋をいくつも通り抜けて彼女の後を追いました。
>
> 魔女は彼女に鍋を掃除し、床を掃き、薪で火を絶やさないように命じました。
```
