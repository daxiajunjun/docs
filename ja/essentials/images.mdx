## 画像

### Markdownを使用する

[Markdown構文](https://www.markdownguide.org/basic-syntax/#images)を使用すると、以下のコードを使用して画像を追加できます。

```md
![タイトル](/path/image.jpg)
```

画像ファイルのサイズは5MB未満である必要があります。それ以外の場合は、[Cloudinary](https://cloudinary.com/)や[S3](https://aws.amazon.com/s3/)のようなサービスにホスティングすることをお勧めします。その後、そのURLを使用して埋め込みます。

### 埋め込みを使用する

画像にもっとカスタマイズ性を持たせるために、[埋め込み](/writing-content/embed)を使用して画像を追加することもできます。

```html
<img height="200" src="/path/image.jpg" />
```

Mintlifyは[Markdown内のHTMLタグ](https://www.markdownguide.org/basic-syntax/#html)をサポートしています。これは、Markdown構文よりもHTMLタグを好む場合に便利で、無限の柔軟性を持つドキュメントを作成できます。

</Tip>

### iFrames

文書内に別のHTMLページを読み込みます。主に動画を埋め込むために使用されます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
