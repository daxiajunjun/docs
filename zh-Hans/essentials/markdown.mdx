## 标题

最适合用作章节标题。

```md
## 标题
```

### 副标题

最适合用作子章节标题。

```md
### 副标题
```

<Tip>

每个 **标题** 和 **副标题** 都会创建一个锚点，并在右侧的目录中显示。

</Tip>

## 文本格式化

我们支持大多数 markdown 格式化。只需在文本周围添加 `**`、`_` 或 `~` 来格式化它。

| 风格         | 写法               | 结果               |
| ------------ | ------------------ | ------------------ |
| 加粗         | `**加粗**`        | **加粗**           |
| 斜体         | `_斜体_`          | _斜体_             |
| 删除线       | `~删除线~`        | ~删除线~           |

您可以将这些组合在一起。例如，写 `**_加粗和斜体_**` 可以得到 **_加粗和斜体_** 文本。

您需要使用 HTML 来编写上标和下标文本。也就是说，在您的文本周围添加 `<sup>` 或 `<sub>`。

| 文本大小     | 写法                       | 结果                     |
| ------------ | -------------------------- | ------------------------ |
| 上标         | `<sup>上标</sup>`         | <sup>上标</sup>         |
| 下标         | `<sub>下标</sub>`         | <sub>下标</sub>         |

## 链接到页面

您可以通过将文本包裹在 `[]()` 中来添加链接。您应该写 `[链接到谷歌](https://google.com)` 来 [链接到谷歌](https://google.com)。

文档中页面的链接需要是根相对的。基本上，您应该包括完整的文件夹路径。例如，`[链接到文本](/writing-content/text)` 链接到我们的组件部分中的页面“文本”。

像 `[链接到文本](../text)` 这样的相对链接会打开得更慢，因为我们无法像处理绝对链接那样轻松进行优化。

## 引用

### 单行

要创建引用，在段落前添加 `>`。

> 多萝西跟随她穿过城堡里许多美丽的房间。

```md
> 多萝西跟随她穿过城堡里许多美丽的房间。
```

### 多行

> 多萝西跟随她穿过城堡里许多美丽的房间。
>
> 女巫让她清洁锅和水壶，打扫地板，并保持火堆有木柴。

```md
> 多萝西跟随她穿过城堡里许多美丽的房间。
>
> 女巫让她清洁锅和水壶，打扫地板，并保持火堆有木柴。
```

### LaTeX

Mintlify 通过 LaTeX 组件支持 [LaTeX](https://www.latex-project.org)。

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
