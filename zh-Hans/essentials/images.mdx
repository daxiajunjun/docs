## 图像

### 使用 Markdown

[Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)让您可以使用以下代码添加图像

```md
![title](/path/image.jpg)
```

请注意，图像文件大小必须小于 5MB。否则，我们建议在像 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 这样的服务上托管。然后，您可以使用该 URL 进行嵌入。

### 使用嵌入

为了获得更高的图像自定义性，您还可以使用 [嵌入](/writing-content/embed) 来添加图像

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

在文档中加载另一个 HTML 页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
